## 🧪 테스트 절차

### Step 1: 스케줄 등록 (1분)

1. **메인 화면** → 우측 상단 **달력 아이콘(📅)** 클릭
2. **스케줄 관리 화면** → 우측 하단 **+ 버튼** 클릭
3. **입력 정보**:
   - **요일**: 오늘 요일 체크 (예: 수요일이면 수요일)
   - **Zoom 링크**: `https://zoom.us/test`
   - **시작 시간**: `현재 시각 + 2분`
     - 예: 지금 08:47 → **08:49** 입력
   - **녹화 시간**: `1` (1분만 테스트)
4. **저장** 버튼 클릭
5. **스케줄 목록에서 확인**

---

### Step 2: 대기 및 관찰 (2분)

#### 2분 대기 중:
- 앱을 그대로 두고 **로그 모니터링**
- Android Studio 터미널 또는 앱 화면 확인

#### 예약 시간 도달 시 예상 동작:

**자동 실행 순서**:
1. **Zoom 실행** (브라우저 또는 Zoom 앱 열림)
2. **녹화 시작** (메인 화면에 진행률 바 표시)
3. **1분 대기**
4. **녹화 종료**
5. **Zoom 종료**

---

### Step 3: 결과 확인 (2분)

#### ✅ 성공 기준

**1. Zoom 실행 확인**
- [ ] 브라우저 또는 Zoom 앱이 자동으로 열렸는가?
- [ ] 로그: `🚀 Zoom 회의 실행 중...`

**2. 녹화 시작 확인**
- [ ] 메인 화면에 녹화 진행률 바가 나타났는가?
- [ ] 로그: `🎬 예약 녹화 시작`

**3. 녹화 파일 생성 확인**
- [ ] 파일 위치: `C:\SatLecRec\recordings\`
- [ ] 파일 이름: `2025-11-06_08-49-00.mp4` (시작 시각 기준)
- [ ] 파일 크기: 1-5MB (1분 녹화)

**4. Zoom 종료 확인**
- [ ] 녹화 종료 후 5초 뒤 Zoom이 자동으로 종료되었는가?
- [ ] 로그: `🚪 녹화 종료 - Zoom 앱 종료 시도`

## 🐛 문제 해결

### 문제 1: Zoom이 안 열림
**증상**: 로그에 `Zoom 실행` 메시지는 있지만 창이 안 열림

**원인**:
- Zoom이 설치되지 않음
- 링크가 잘못됨

**해결**:
1. Zoom 설치 확인: https://zoom.us/download
2. 실제 Zoom 미팅 링크 사용

---

### 문제 2: 녹화가 안 시작됨
**증상**: Zoom은 열렸지만 녹화가 시작 안 됨

**로그 확인**:
```
❌ 녹화 시작 실패
❌ Audio 장치를 찾을 수 없습니다
```

**해결**:
- 오디오 장치(마이크, 스피커) 연결 확인
- Windows 사운드 설정 확인

---

### 문제 3: 스케줄이 실행 안 됨
**증상**: 예약 시간이 지났는데 아무 일도 안 일어남

**원인**:
- 시간 설정이 과거 시각
- 요일이 잘못 설정됨
- 스케줄이 비활성화됨

**해결**:
1. 스케줄 목록에서 스케줄 상태 확인
2. 시간 다시 설정 (현재 + 2분)
3. 요일 확인 (오늘 요일이 체크되어 있는지)
