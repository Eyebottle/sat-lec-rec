## ⏰ 테스트 F: 예약 스케줄 실행 (최종 검증)

> **목적:** 실제 예약 시간에 자동 실행되는지 확인  
> **소요 시간:** 약 5~10분

### 테스트 절차

1. **스케줄 생성** (현재 시간 + 3분 후)
   - 스케줄 이름: `테스트 스케줄`
   - 요일: 오늘 요일 선택
   - 시작 시각: 현재 시간 + 3분 (예: 지금 08:05이면 08:08로 설정)
   - 녹화 시간: 1시간 (테스트니까 실제로 1시간 기다릴 필요 없음)
   - Zoom 링크: 테스트 링크 입력
   - ✅ 활성화 상태로 저장

2. **대기 및 관찰**
   - 앱 메인 화면에서 "다음 예약" 정보 확인
   - 예약 시간까지 대기 (손 떼고 지켜보기)

| # | 테스트 항목 | 예상 결과 | 결과 | 비고 |

| F1 | 스케줄 추가 완료 | 카드에 "다음: 오늘 HH:MM" 표시 | ⬜ | |
| F2 | AppBar에 다음 예약 정보 표시 | 남은 시간 카운트다운 | ⬜ | |
| F3 | 예약 시간 도달 시 자동 실행 | Zoom 자동 열림 | ⬜ | ⚠️ 핵심! |
| F4 | 녹화 시작 알림 | 시스템 트레이 알림 | ⬜ | |
