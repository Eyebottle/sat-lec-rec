# 트레이 아이콘

트레이 아이콘 파일을 이 폴더에 배치하세요.

## 필요한 파일
- `tray_icon.png` (32x32 또는 64x64 픽셀 권장)
- 또는 `tray_icon.ico` (16x16, 32x32 크기 포함)

## 다운로드 사이트
- Flaticon: https://www.flaticon.com/free-icons/camera
- Icosix: https://www.icosix.com/icon/259/camera
- Icon-Icons: https://icon-icons.com/search/icons/camera

## 임시 해결책
현재는 빈 아이콘 경로로 설정되어 있어 Windows 기본 아이콘이 사용됩니다.
